The most lightweight Google reCAPTCHA v2 comment anti-spam plugin!

== Description ==
This plugin will add a Google reCAPTCHA v2 challenge to your comment form.
Just add your free keys and your good to go.
Kills 99% of all known comment spam... DEAD!
					
== Installation ==
The easiest way to enjoy LDW reCAPTCHA is to login to your WordPress dashboard, go to Plugins > Add New,
search for LDW reCAPTCHA and click 'Install Now'.

You can also download the zip file from this page and upload it from the Plugins > Add New > Upload page.

== Frequently Asked Questions ==
= Usage =
1. Dashboard > Settings > LDW reCAPTCHA.
1. Enter your keys.
1. Click 'Save Changes'.

= It's Not Working! =
* It does not show the reCAPTCHA to logged in users.
* It does not show the reCAPTCHA if valid keys have not been entered.
* It does not show the reCAPTCHA if it cannot connect to Google's servers.

= I Still Get Spam! =
* If someone is actually visiting your site to post spam, then there is not a lot you can do about it.
This plugin checks for spam posted by bots (the vast majority of spam).
* It marks comments as spam if the reCAPTCHA test is failed, it does not just nuke the comment.
* It does not check pingbacks and trackbacks.
